Mini E-Commerce Tester

Automated testing project using Selenium WebDriver, designed to validate core workflows of a mini e-commerce website.
This project focuses on writing clean, reusable, and maintainable test scripts with proper reporting and screenshot capturing.

Features
Automated Functional Tests

User login & authentication

Product search

Add to cart functionality

Checkout and order flow

Validating UI elements

Page navigation tests

Selenium-Based Automation

Uses Selenium WebDriver

Browser automation (ChromeDriver )

XPath & CSS selectors

Reporting & Screenshots

Captures screenshots on test failures

Run Command : Pytest


Stores results in a reports and screenshots folder

Works with pytest (if you are using Python)
